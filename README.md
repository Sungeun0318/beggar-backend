# beggar-backend

거지 우정 수호대 (Save My Friendship) - Spring Boot 백엔드 API

## Tech Stack
- Java
- Spring Boot
- Spring Security + JWT
- MySQL 8.0
- AWS (EC2, RDS, S3)

## Features
- 카카오 OAuth2 로그인 및 JWT 인증
- 거지방 CRUD 및 초대 링크/코드 관리
- 익명 예산 입력 및 접근 격리
- 추천 요청 중계 (Python AI 서버 연동)
- 영수증 업로드 (S3) 및 지출 관리
- 소득공제 시뮬레이션 데이터 제공
- FCM 푸시 알림
